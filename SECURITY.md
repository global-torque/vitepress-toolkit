# Security Policy

## Supported versions

| Version               | Security fixes                                   |
| --------------------- | ------------------------------------------------ |
| Latest `0.2.0-beta.*` | Yes, after its protected prerelease is published |
| Older `0.2.0-beta.*`  | No; upgrade to the latest reviewed prerelease    |
| `0.1.x` and older     | No                                               |

Report security issues for the public package through:

- GitHub: https://github.com/global-torque/vitepress-toolkit/security/advisories/new

Do not include secrets, private content, customer data, or unpublished
vulnerability details in public issues.

Maintainers target acknowledgement within three business days and an initial
triage response within seven business days. Remediation and coordinated
disclosure timing depend on severity and whether an affected immutable artifact
exists.
